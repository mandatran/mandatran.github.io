Thought­STEM Internship

Created and documented Kata Collections, which are curriculum to teach kids how to code. Helped scaffold learning by constructing problem sets that build on the previous exercise and increases in difficulty. My main contributions were to the Data Science, Virtual Reality, and Fundamental Kata Collections.

Timeline

June 2019 - August 2019

Tools & Technologies

Racket, Scribble

Role

Software Engineer Intern and Computer Science Educator

</> Code

Takeaways

This was my first time internship experience and I am so grateful I had the opportunity to help further early computer science education by creating curriculum. I was also able to inspire seventh to tenth graders to pursue computer science by teaching them how to code games they were excited to play and later how to create games by themselves.

What is ThoughtSTEM

ThoughtSTEM is a computer science education company in San Diego. They teach students how to code at early age by creating engaging learning environments (such as coding playable games).

Links

01. Data Science Kata Collection

The Data Science Kata Collection aimed to teach the basics of data analytics and how to use Racket to create simple data visualizations.

02. Virtual Reality Kata Collection

The Virtual Reality Kata Collection is a more advanced set of problems that will allow students to apply their knowledge from the Fundamental Kata Collection to larger challenges and create their own virtual reality.

03. Fundamental Kata Collection

The Fundamental Kata Collection aimed to teach basic coding skills visually by using the 2htdp/image and universe Racket libraries to build simple images and animations.

04. My Blog

Blog that was kept to document the work I completed each day during this internship. Wrote down the major tasks, what I had learned, and my takeaways. This was completed using Scribble (a Racket documentation tool).

05. ThoughtSTEM 2019 Repo

Repository that contains all the intern blogs and Kata Collections created in the summer of 2019.
