About


Hi I’m Manda, and I really like watching TV. I am very interested in cognitive science particularly in regard to machine learning. This website demonstrates how machine learning algorithms can be applied to a Netflix dataset in order to recommend titles based on a single movie/tv show. My goal is to hopefully inspire people, especially new cognitive science majors, to learn more about recommendation systems and machine learning in general.

This website was created for the course CAT 125: Public Rhetoric and Practical Communication. The purpose of this website is to explore and showcase a particular interest, mine being the intersection of machine learning and entertainment. You will find the “Links” and “Critique” pages under the tab labeled “Discussion” in the top navigation bar. To understand my process of creating this project, please check out the “Process” page in the top navigation bar. Here I will detail my inspiration as well as describe the step-by-step evolution of this project. To check out the code used to create the recommendation system you can either check out my Jupyter Notebook in the “Process” or check out the code on my GitHub.

I have more exciting projects in the works, including building an interactive web-application to showcase this project in real-time.
