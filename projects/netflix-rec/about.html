Hi I’m Manda, and I really like watching TV. You can learn more about me here. But to sum it up, I am very interested in cognitive science particularly in regards to machine learning. This website demonstrates how machine learning algorithms can be applied to a Netflix dataset in order to recommend titles based on a single movie/tv show. My goal is to hopefully inspire people to learn more about recommendation systems and machine learning in general.
